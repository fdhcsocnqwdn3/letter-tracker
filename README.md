# Letter Tracker

A simple web app for tracking letter knowledge during early reading education, following Erik Hoel's phonics method.

## Features

- Track uppercase and lowercase letters separately
- Red/Amber/Green progress indicators for phonics (letter sounds)
- Letters ordered by frequency in English language
- Progress statistics and history tracking
- Export/import data to sync between devices
- Mobile-first responsive design

## Usage

Visit the hosted app or open `index.html` in any modern web browser.

## Data Privacy

All data is stored locally in your browser. Your progress is private and never sent to any server. Use the export/import feature to sync data between devices manually.

## Based on

Erik Hoel's teaching method as described in "The BIG GUIDE to teaching LITTLE PEOPLE how to sound out words"
